# NQueens Genetic Algorithm

A Java project for solving the N-Queens problem using genetic algorithms.

## Getting Started

### Prerequisites
- Java 11 or higher
- Maven 3.6 or higher

### Building the Project
```bash
mvn clean compile
```

### Running the Application
```bash
mvn exec:java -Dexec.mainClass="com.example.App"
```

### Running Tests
```bash
mvn test
```

## Project Structure
- `src/main/java/` - Main source code
- `src/test/java/` - Test source code
- `pom.xml` - Maven configuration
